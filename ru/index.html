Как разблокировать пользователей Tor
на своем сайте, защищенном Cloudflare

Что такое Cloudflare?

Cloudflare — CDN сервис, работающий как реверс прокси для сайтов. Он защищает от вредоносных запросов и оптимизирует доставку контента путем отдачи кэшированного контента со своих серверов. Если вы используете Cloudflare, эта статья поможет добавить в белый список запросы из сети Tor. Если ваш любимый сайт использует Cloudflare и вы испытываете трудности с доступом к нему через Tor — отправьте ссылку на эту статью администраторам сайта.

Что такое Tor?

Tor — свободное программное обеспечение для анонимизации. Tor использует луковую маршрутизацию чтобы перенаправлять трафик через случайно выбранные узлы, находящиеся в разных странах, тем самым защищая пользователей по всему миру от цензуры и массовой слежки. На сайте Tor Project размещен список историй о том, как люди используют Tor для благих целей.

Зачем мне делать это?

Cloudflare использует систему репутации IP адресов, которая оценивает их основываясь на истории трафика. Tor использует ограниченное количество выходных узлов (это значит что один IP адрес используется большим количеством людей, большинство из которых отправляет невредоносные запросы), и это причина, по которой Cloudflare блокирует по умолчанию большинство узлов. Пользователи Tor Browser решают десятки каптч в день из-за популярности Cloudflare и механизма Tor Browser по изолированию цепочек.

Это выглядит так.

Исследование показывает, что как минимум 80% выходных узлов Tor блокируются Cloudflare. Исследование конкурента Cloudflare, Akamai, показывает что пользователи Tor имеют практически одинаковый показатель конверсии (1:833 против 1:895) по сравнению с другими пользователями. Вывод Akamai: «Многие исследовательские и новостные публикации показывают, что сеть Tor несет большой спектр рисков, но в то же время, большинство полностью обходят стороной факт того, что существует также потенциал для бизнеса в позволении пользователям Tor использовать сайты, приносящие доход».

Как добавить пользователей Tor в белый список?

  1. Войдите в аккаунт Cloudflare

  2. Выберите домен

  3. Откройте вкладку «Firewall»

  4. Прокрутите вниз до настройки «Access rules»

  5. Найдите «Tor», выберите действие (Whitelist) и укажите должно ли это правило использоваться только для этого сайта (This website) или для всех ваших сайтов (All your websites)

  6. Готово
